# Innovation Labs 2026 – Bu Website

Official landing page for **Bu – Newborn's Quiet Guardian**, prepared for the **Innovation Labs 2026** edition.

Bu presents a calm, minimalist digital experience for newborn health tracking, AI-powered insights, and pediatric-ready report sharing.

## Project Overview

This repository contains a single-page website that showcases:

- The product value proposition
- Core features of the Bu app
- A simple 3-step usage flow
- Parent testimonials
- Mobile-first, responsive presentation

## Tech Stack

- **HTML5** (single-page structure)
- **Tailwind CSS (CDN)** for styling and layout
- **Vanilla JavaScript** for dynamic section rendering and interactions
- **Font Awesome** icons
- **Google Fonts (Montserrat)**

## Repository Structure

- `main.html` – Complete landing page markup, styles, and scripts
- `img-1.png` – Product preview image
- `img-2.png` – Product preview image

## Key Features

- Sticky responsive navigation with mobile menu toggle
- Hero section with strong product messaging
- Visual app preview block
- Dynamic rendering for:
  - Navigation links
  - Feature cards
  - “How It Works” steps
  - Testimonials
  - Footer links
- Smooth scrolling between page sections
- Section-aware active navigation highlighting

## Run the Website

No build or dependency installation is required.

### Quick Start

Open `main.html` directly in any modern browser.

### Recommended (Local Server)

If Python is available:

```bash
python -m http.server 8000
```

Then visit:

`http://localhost:8000/main.html`
